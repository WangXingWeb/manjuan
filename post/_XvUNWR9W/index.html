ant-design中使用iconfont图标

2019-02-05

ant-design中默认提供了很多图标，但是内置的图标毕竟有限，很多时候我们还是需要自己在iconfont中找一些图标，iconfont.cn和ant-design都是阿里的产品，所以ant-design给我们提供了使用iconfont的方法。

import { Icon } from "ant-design-vue";
const IconFont = Icon.createFromIconfontCN({
    scriptUrl: "//at.alicdn.com/t/font_1750375_zlsnz3xoaaj.js"
});
export default IconFont

需要注意的是我在scriptUrl中配置的是线上iconfont提供的地址，相当于iconfont将生成的图标js文件放到了iconfont的静态资源服务器上，我们直接应用这个地址就可以了。你也可以把它下载到本地，应用本地的js文件，应用本地的js文件需要每次添加或者修改图标后在下载图标文件到本地。直接应用地址只需要重新生成在线链接，替换scriptUrl就可以了，会方便一些。
