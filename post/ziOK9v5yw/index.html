Vue Router导航守卫

2020-01-06
const router = new VueRouter({
  routes
})

router.beforeEach((to, form, next) => {
  if (!to.meta.isPublic && !localStorage.token) {
    return next('/login')
  }
  next()
})

上面的代码我们在路由发生改变后在页面跳转前去验证localStorage是否有token，也就是验证用户是否已经登录，同时判断即将跳转的路由是否是开放的页面（公开的，用户不登录也能访问），如果目标路由不是可以公开访问的同时用户也没有登录则直接跳转到login页。
