axios配置默认参数

2020-03-25

正常开发中很多请求都需要携带上用户的token等登录凭证才能够请求数据，每次发起ajax请求都写一次会非常麻烦。这里我们使用axiox拦截器，拦截请求注入token，这样每次发起的ajax就会默认带有token参数了。

http.interceptors.request.use(function (config) {
    if (localStorage.token) {
        config.data += '&token=' + localStorage.token
    }
    return config
}, function (error) {
    return Promise.reject(error)
})
