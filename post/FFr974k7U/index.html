快捷键记录

2020-01-06

现在大多数的快捷键都支持自定义了。所以这里记录的快捷键都为默认的快捷键。

浏览器快捷键

以下快捷键适用于chrome浏览器，其他浏览器可能有部分快捷键是不适配的。

  • 切换标签页

    1. 从左向右切换 ctrl+tab
    2. 从右向左切换ctrl+shift+tab

  • 新建标签页

    1. mac中 command+T
    2. windows中ctrl+T

  • 关闭标签页

    1. mac中 command+W
    2. windows中ctrl+W

  • 空格键可以翻屏

  • 光标快速移动到地址栏

    1. mac command+L
    2. windowsctrl+L

  • 清除缓存 command+shift+delete

系统快捷键

  • 隐藏当前窗口
    1. mac中 command+H
    2. windows中ctrl+H

通用

  • siri快捷键按住command+空格

  • 输入法（两种情况）

    1. 默认option+空格切换输入法，caps lock切换大小写

    2. 经过如下图的设置之后，caps lock也可以切换输入法，长按caps lock设定为锁定大写

  • 快速打开表情与符号 command+ctrl+空格

  • 强制退出应用程序 option+command+esc

  • 退出应用程序 command+Q

  • 快捷显示qq界面 ctrl+command+q

  • qq截图 ctrl+command+A

  • qq录屏 ctrl+command+r

  • 快捷显示qq界面 ctrl+command+w

  • 最小化窗口

    1. command+m带动画
    2. command+h不带动画

  • 隐藏其他窗口 command+option+h

  • 显示正在运行的应用程序 command+tab，按住command不松，按tab或者左右键可切换应用程序

  • 全屏显示ctrl+command+f

  • 显示桌面command+f3或者fn+f11

文件管理

  • Command + Shift + N 新建文件夹（New）

  • Command + Shift + G 调出窗口，可输入绝对路径直达文件夹（Go）

  • return 这个其实不算快捷键，点击文件，按下可重命名文件

  • Command + O 打开所选项。在Mac里打开文件不像Windows里直接按Enter

  • Command + Option + V 作用相当于Windows里的文件剪切。
    在其它位置上对文件复制（Command-C），在目的位置按下这个快捷键，文件将被剪切到此位置

  • Command + ↑ 打开包含当前文件夹的文件夹，相当于Windows里的“向上”

  • Command + Delete 将文件移至废纸篓

  • Command + Shift + Delete 清倒废纸篓

  • Space 快速查看选中的文件，也就是预览功能

截图

  • Command + Shift + 3 截取全部屏幕到文件

  • Command + Shift + Control + 3 截取全部屏幕到剪贴板

  • Command + Shift + 4 截取所选屏幕区域到一个文件，或按空格键仅捕捉一个窗口

  • Command + Shift + Control + 4 截取所选屏幕区域到剪贴板，或按空格键仅捕捉一个窗口

文本处理

  • Command + B 切换所选文字粗体（Bold）显示
  • fn + Delete 相当于PC全尺寸键盘上的Delete，也就是向后删除
  • fn + ↑ 向上滚动一页（Page Up）
  • fn + ↓ 向下滚动一页（Page Down）
  • fn + ← 滚动至文稿开头（Home）
  • fn + → 滚动至文稿末尾（End）
  • Command + → 将光标移至当前行的行尾
  • Command + ← 将光标移至当前行的行首
  • Command + ↓ 将光标移至文稿末尾
  • Command + ↑ 将光标移至文稿开头
  • Option + → 将光标移至下一个单词的末尾
  • Option + ← 将光标移至上一个单词的开头
  • Control + A 移至行或段落的开头
