a++和++a的区别

2016-09-16
let a = 1;
let b = a++;
console.log(b); //1

上面的代码可以看出a++是先返回后+1,也就是先将a的值赋值给b，然后a=a+1

let a = 1;
let b = ++a;
console.log(b);//2

++a的过程是相反的，先执行a=a+1,然后将a的值赋值给b，所以得到的b的值为2

同样a--和–a也是同样的情况

let a = 1;
let b = a--;
console.log(b);//1

let a = 1;
let b = --a;
console.log(b);//0
