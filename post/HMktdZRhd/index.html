vue中watch某个对象的属性

2018-05-18
data:{
	userInfo：{
		name:'',
		age:'',
		...
	}
}

如果我们要监听userInfo中的name,如果我们监听整个userInfo当userInfo中数据很多时，占用很多资源，影响程序的性能。

watch:{
    userInfo(val,oldVal){
        //do something about val.name
    }
}

上面的代码不论userInfo中的那个属性发生变化，都会执行这段代码。而本质上我们只想监听userInfo中的name,其它属性改变比如age是我们不关心的。这样就造成了额外开销。

其实vue给我们提供了单独监听某个对象的某个属性的方法

watch:{
    "userInfo.name"(val){
        //do something about val
    }
}

这样我们单独监听userInfo的name属性了
