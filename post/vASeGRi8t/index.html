vue父组件触发子组件的事件

2018-09-01

父组件代码

<template>
  <div>
    <button @click="clickParent">点击</button>
    <child ref="mychild"></child>
  </div>
</template>
 
<script>
  import Child from './child';
  export default {
    name: "parent",
    components: {
      child: Child
    },
    methods: {
      clickParent() {
        this.$refs.mychild.$emit("childEvent",'发送的数据');
      }
    }
  }
</script>

关键在于使用 $refs找到子组件然后使用 $emit触发子组件绑定的事件

子组件代码

<template>
  <div>
    child
  </div>
</template>
 
<script>
  export default {
    name: "child",
    created() {
      this.$on("ChildEvent", (res) => {
        this.childFun(res);
      });
    },
    methods: {
      childFun(res) {
        console.log(res)
      }
    }
  }
</script>

子组件使用$on绑定事件
