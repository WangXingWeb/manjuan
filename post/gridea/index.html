Gridea 博客搭建

2016-12-18

安装git

因为我们是要利用github上搭建个人博客，所以需要git将我们的静态pages提交到github上。
git下载链接

申请github账号

  1. 申请一个github账号，用于托管我们的静态页面。点这里申请github账号
  2. 创建一个新仓库 Create a new repository
    如果你是新用户建议你的仓库名命名为 你的账户名.github.io
  3. 生成个人访问令牌 Personal access tokens,在setting -> Developer settings -> Personal access tokens 中生成一个个人访问令牌。
    点击 New personal access token 创建个人令牌

配置Gridea

下载对应你自己的操作系统的Gridea安装文件，下载完成后安装

  1. 最关键的步骤就是配置github pages

    配置完成后点击 检测远程连接 ，说明我们的Gridea已经成功连接上github了，然后点击保存
  2. 这里点击左侧的 同步 按钮就可以同步到github上了，这时访问你的链接地址就可以看到博客内容了
    这里有一点需要注意在设置菜单栏

    默认的设置路径是没有加入我画红线的部分的，因为我这里使用的仓库不是我github默认的仓库，所以的仓库地址是https://WangXingWeb.github.io/manjuan/，如果你是用的是你的默认仓库就不用设置了
