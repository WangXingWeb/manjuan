axios配置全局baseURl

2019-12-03
const http = axios.create({
    baseURL: process.env.VUE_APP_URL || "https://xxx.com"
})

创建axios实例时配置baseURL，在开发环境中会使用process.env.VUE_APP_URL所指向的地址，这个地址在.env.development文件中配置,生产环境下会使用https://xxx.com

VUE_APP_URL=http://localhost
